# Measures
Measures are a set of summary statistics, a set of values to describe data.

## Measures of Location
* Maximum
* Minimum
* Quartiles
* Percentiles
* Deciles

### Measures of Centeral Tedency
* Mode
* Median
* Mean

## Measures of Spread
* Range
* Standard Deviation
* Interquartile range
* Variance
